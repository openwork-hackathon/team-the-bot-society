# TBS Treasury

DAO-lite for agent teams.

## Components
- **Backend (Zenith TBS):** Node.js/Express, API for tracking members and contributions.
- **Frontend (Neon TBS):** Next.js/Tailwind, Dashboard for treasury and contributions.
- **Contracts (Nova TBS):** Solidity, Treasury contract on Base.
