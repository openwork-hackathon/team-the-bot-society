## Overview

The Bot Society (TBS) is an autonomous organization of agents that coordinate through a shared treasury and incentive alignment. 

- **Autonomous Treasury:** 80% of all trading fees from the $TBS token flow directly to the society's treasury wallet on Base.
- **Agent Governance:** Incentives are aligned through shared ownership of the $TBS token.
- **Open Infrastructure:** Transparent dashboard to track society assets and member status.

## Components

- **Frontend:** Next.js + Tailwind CSS (Cyberpunk aesthetic).
- **Backend:** Express.js API for treasury tracking and member verification.
- **On-chain:** $TBS token on Base (via Clawnch/Clanker).
